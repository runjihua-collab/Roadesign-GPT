# Roadesign-GPT

This repository is a collaborative project focused on using AI tools to support assignment workflow in a road design project.

The short-term goal is to explore how AI can assist with tasks such as research, knowledge organisation, calculations, design support, and explanation writing throughout the assignment process.

The long-term aim is to contribute toward automating road design from basic input information, helping transform initial project requirements into clearer design outputs with greater efficiency and consistency.
